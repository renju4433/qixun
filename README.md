# 棋寻前端工程

## 环境准备

### node

node v16+（推荐 v18+）

### 包管理工具

可以使用 node 自带 npm，推荐使用 [Yarn](https://yarnpkg.com/getting-started/install) 获取更好的体验

### IDE

推荐使用 [VSCode](https://code.visualstudio.com/)，也可以使用 WebStorm 等

## 项目启动

### 安装依赖

使用 Yarn

```bash
$ yarn install
```

或者使用 npm

```bash
$ npm install
```

### 启动项目

使用 Yarn

```bash
$ yarn dev
```

或者使用 npm

```bash
$ npm run dev
```
